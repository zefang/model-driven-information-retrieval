DocuWare KALKULATIONSDISKETTE
-----------------------------

Sehr geehrter Interessent,

vielen Dank für die Anforderung der DocuWare Kalkulations-
diskette. Sicherlich sind Sie neugierig, was die elektronische
Archivierung in Ihrem Unternehmen an Einsparungen bringen kann!
Um Ihnen möglichst schnell einen Überblick über die Möglichkeiten
dieser Kalkulationsdiskette zu bieten, hier nun einige Infor-
mationen:


Excel-Datei:
------------
  Die Datei KALKULAT.XLS ist eine Microsoft Excel für Windows
  Datei.
  Sie bietet Ihnen die Möglichkeit, an Hand Ihrer individuellen
  Daten einen Kostenvergleich zwischen der Papierablage und der
  elektronischen Archivierung zu erstellen.

Kalkulations-Prämissen:
-----------------------
  Tragen Sie dazu in die Felder des Bereichs "Prämissen" Ihre
  unternehmensspezifischen Werte ein. Die vorgegebenen Angaben sind
  Erfahrungswerte, die jedoch nicht mit Ihren Unternehmensdaten
  korrespondieren müssen.
  Mit den Prämissen können folgende Optionen variabel gewählt
  werden:

  - Arbeitstage/Jahr: effektive Arbeitstage pro Jahr.

  - Personalkosten DM/Std.: durchschnittliche Personalkosten
    aller im Ablage- und Rechercheprozeß beteiligter Personen.

  - Sortierleistung Blatt/Std: Dokumente müssen zum Ablegen
    i.d.R. vorsortiert werden. Anzahl der Blatt, die pro Stunde
    sortiert werden können (abhängig von Komplexitätsgrad der
    Sortierkriterien).

  - Ablageleistung Blatt/Std: Anzahl der Blatt, die pro Stunde
    in Ordner abgelegt werden können.

  - Sachmittelkosten in DM: durchschnittliche Kosten pro 100 Blatt 
    für Aktenordner, Hefter, etc. bei Papierablage.

  - Recherchezeit: durchschnittliche Recherchezeit  und
    Wiedereinsortierzeit (bei Belegentnahme) für einen Beleg in
    der Papierablage.

  - Scannen Seiten/Minute: durchschnittliche Scannleistung pro
    Minute; ist stark abhängig von Scannerklasse (zwischen 8 und 
    100 Seiten pro Minute) und Papierqualität.

  - Manuelles Indizieren Blatt/Stunde: Indizierungsleistung pro
    Stunde, ist abhängig von Anzahl der zu vergebenden 
    Suchbegriffe.

  - Sachmittelkosten: Kosten pro 100 Blatt für optische Platte,
    Preis z.Z. für eine CD-ROM ca. 15.- DM.

  - Seitenkapazität auf CD-ROM (640 MB): durchschnittliches
    Speichervolumen in Seiten (abhängig von Scanvorlagen,
    durchschnittlicher Speicherbedarf pro Seite bei kaufmännischen
    Unterlagen: 50 KB)

  - Recherchezeit Elekt. Ablage (in min): durchschnittliche
    Recherchezeit in einem elektronischen Ablagesystem, ist
    abhängig von Speichermedium (Festplatte, optische Platte, 
    Jukebox).

  - Neubeleganfall pro Tag: Anzahl der Belege, die pro Tag neu
    erfaßt werden sollen. Es können fünf verschiedene Mengen auf
    einmal durchgerechnet werden.

  - Recherche pro Tag: Durchschnittliche Anzahl der Zugriffe
    aller Mitarbeiter auf die abgelegten Dokumente pro Tag.


Kalkulation:
------------
  Die Daten werden in die Kalkulationen für die Arbeitsvorgänge
  ABLAGE und RECHERCHE übernommen und vergleichend ausgewertet. Die
  integrierten Säulendiagramme zeigen Ihnen anschaulich den
  jeweiligen Kostenverlauf.

  Im abschließenden Vergleich GESAMTKOSTEN-EINSPARUNG werden die
  beiden Vorgänge kostenmäßig zusammengefaßt und die Kosten-
  differenzen zwischen den alternativen Methoden Papierablage und
  elektronische Archivierung in Abhängigkeit von abzulegendem
  Belegvolumen und Recherchehäufigkeit dargestellt. Die
  Einsparungen beziehen sich auf ein effektives Arbeitsjahr.
